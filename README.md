# Solar Income Forecaster – AI-Powered Solar Investment Planner ☀️

**Real NASA satellite data + Deep Learning (LSTM) forecast → Accurate 25-year profitability analysis**

A beautiful, fully functional Streamlit web app that:
- Fetches 40+ years of real NASA solar radiation data (GHI) for any location on Earth
- Trains (or loads) an LSTM neural network to predict the next 5 years of monthly solar irradiance
- Calculates detailed cash flows, payback period, and 25-year profit for up to 3 different solar system sizes
- Visualizes historical + AI-forecasted sunlight and cumulative profit over time

Perfect for homeowners, solar installers, consultants, and investors in India (or anywhere).

### Features
- Real-time NASA POWER API integration
- On-the-fly LSTM training (caches model locally)
- 5-year AI sunlight forecast with trend percentage
- Indian rupee (₹) based financial modeling with tariff inflation
- Compare 3 system configurations side-by-side
- Gorgeous charts (just like the screenshot!)
- 100% offline-capable after first run (model saved)
